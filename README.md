📊 California Labor Market Analysis (1990-2024)

An advanced Power BI Dashboard that moves beyond static reporting into **Statistical Forecasting** and **AI-Driven Risk Analysis**.

## 🚀 Key Features
- **Macro Trends:** Tracking 30+ years of employment cycles (including 2008 & 2020 crises).
- **Statistical Intelligence:** Used **Standard Deviation** and **Variance** to identify economic volatility.
- **AI-Powered Insights:** Integrated **Key Influencers** visual to find hidden triggers of unemployment.
- **Interactive UI:** Glassmorphism theme with custom page navigation.

## 💡 Top Insights
1. **Regional Divide:** Coastal tech hubs vs. Inland agricultural zones.
2. **Shock Sensitivity:** Rapid recovery patterns post-pandemic.
3. **Volatility:** High human burden and risk in major metropolitan hubs like LA.

## 🛠️ Tools Used
- Power BI Desktop
- DAX (Calculated Measures)
- Power Query (ETL)
- Statistical Modeling
